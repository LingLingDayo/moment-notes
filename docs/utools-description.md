## 一句话介绍

高颜值便签，双击一键极速粘贴。

## 详情介绍

一款专为 uTools 打造的便签插件小工具。支持分类管理、多维排序、彩色卡片、自定义标签等功能，助你清爽、高效地记录日常灵感与备忘。

🌈 功能特色

双击粘贴：非编辑状态下双击卡片可一键隐藏并粘贴文本到光标处，效率翻倍。
分类管理：支持无限层级的子分类与自由排序，且支持快速添加子分类、重命名与删除。
超级面板：集成 uTools 超级面板，支持划词一键捕获文本极速生成卡片。
多维检索：支持“标题/内容/标签”范围精准模糊搜索，查找定位更快捷。
灵活排序：支持按时间、拼音、标签等属性自动升降序，也支持拖拽手动排序。
快捷键操作：支持快捷新建、一键聚焦搜索框、编辑保存与取消等全局快捷键操作。
备份与同步：支持单卡片TXT导出、全局JSON备份，在uTools环境下自动加密云同步。
