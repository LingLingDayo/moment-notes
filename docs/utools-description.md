## 一句话介绍

高颜值便签工具，极速粘贴备忘。专为 uTools 打造的极简记事本、高效备忘录与待办清单，支持快捷回复、快捷输入与划词捕获。

## 详情介绍

一款专为 uTools 打造的高颜值便签工具。集记事本、备忘录与待办清单于一体，支持快捷回复、快捷输入与划词捕获。提供多层级分类管理、多维排序、彩色卡片与自定义标签等丰富功能，助你清爽、高效地记录日常灵感与备忘。

🌈 功能特色

双击粘贴：非编辑状态下双击卡片可一键隐藏并粘贴文本到光标处，效率翻倍。
分类管理：支持无限层级的子分类与自由排序，配备“最近使用”高频分类与“最近删除”回收站。
超级面板：集成 uTools 超级面板，支持划词一键捕获文本极速生成卡片。
多维检索：支持多关键词与“标题/内容/标签”范围精准&模糊搜索，查找定位更快捷。
灵活排序：支持按时间、拼音、标签等属性自动升降序，也支持拖拽手动排序。
快捷键操作：支持自定义全局快捷键，支持快捷新建、一键聚焦搜索框、编辑保存与取消等操作。
个性化定制：内置设置中心，支持自定义工具栏、卡片外观、日期格式、悬浮动效、快捷键等多种偏好设置。
备份与同步：支持全局JSON备份，在uTools环境下自动加密云同步。
